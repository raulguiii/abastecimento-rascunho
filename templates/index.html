📌 Abastecimento / Instruções

🚗 Bem-vindo ao Sistema de Abastecimento da SEMECTI

Este site foi desenvolvido para registrar e organizar todos os abastecimentos de forma eficiente e transparente. Para garantir o uso correto, siga atentamente as instruções abaixo.

📋 Como Funciona?

O sistema é dividido por departamentos, e cada um possui sua própria aba e cadastro. Os funcionários somente podem acessar a aba correspondente ao seu departamento, então fique atento a essa regra.

🔑 Como Criar uma Conta?

Para obter acesso ao sistema, entre em contato com Raul e envie os seguintes dados:

  • ✅ Nome completo
  • ✅ RGF
  • ✅ Cargo
  • ✅ Departamento
  • ✅ CPF

⚙️ Funcionalidades do Sistema

📝 Autorização de Abastecimento

Após abastecer, você deve preencher os seguintes dados:

  • 📌 Nome
  • 📌 RGF
  • 📌 KM atual do veículo
  • 📌 Placa do veículo
  • 📌 Data do abastecimento
  • 📌 Nome do posto
  • 📌 Quantidade de litros
  • 📌 Valor total (R$)
  • 📸 Foto do comprovante de pagamento
📂 Histórico de Abastecimentos

Na aba histórico você pode visualizar todos os abastecimentos do mês atual, com detalhes e acesso aos comprovantes. Os comprovantes ficam disponíveis somente até o final do mês. Após esse período, os registros são transferidos para uma aba exclusiva, acessível apenas pelos diretores de departamento.

🔔 Mantenha suas informações sempre atualizadas e siga as diretrizes para o bom funcionamento do sistema.

❓ Caso tenha dúvidas, entre em contato com o responsável pelo seu departamento.
